# Program analisis data

`export_experiment_csv.py` mengekspor data yang diperlukan untuk mereplikasi analisis lima kelompok eksperimen. Program memakai pustaka standar Python dan tidak memerlukan pandas.

Contoh penggunaan dari folder `code`:

```powershell
python analysis/export_experiment_csv.py ..\results\tugasakhir.db
```

Gunakan argumen `--output` jika CSV perlu disimpan pada folder lain.
